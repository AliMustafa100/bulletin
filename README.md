# Job Tracker

A simple, user-friendly website to organize your job applications. Paste job descriptions, tailored resumes, and CVs — then add them to your board as organized columns.

## Features

- **Paste & store** — Drop in job descriptions, tailored resumes, and tailored CVs (all optional)
- **Kanban board** — Each application becomes its own column on a horizontal board
- **Edit anytime** — Update any application after adding it
- **Auto-save** — Everything is saved locally in your browser (no account needed)
- **Expand/collapse** — Show or hide content within each column to keep things tidy

## Getting Started

```bash
npm install
npm run dev
```

Open the URL shown in your terminal (usually `http://localhost:5173`).

## Build for production

```bash
npm run build
npm run preview
```

## Tech

- React + TypeScript
- Vite
- LocalStorage for persistence
